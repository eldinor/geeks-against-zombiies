Secure the network

Networks carry identities, private conversations, payments, and the services people rely on. If attackers break through, they can steal data, disrupt work, or use one compromised device to reach many others.

Real security uses layers: timely updates, strong access controls, monitoring, backups, and people who know how to recognize threats. In this game, each geek represents one part of that shared defense.

Build a resilient team and stop every threat before it crosses the network.

How to play

Select a geek, then place them on an empty tile. Stop every zombie before it reaches the left edge.

1–3 select · Space pauses · arrow keys move the cursor · Enter places
